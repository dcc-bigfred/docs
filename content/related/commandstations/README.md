# Command stations

Hardware reference for physical **command stations** supported or
commonly used with BigFred. For how BigFred wires a station into the hub
(catalogue row, `dcc-bus` daemon, layout attachment), see
[`16-dcc-bus`](../../specs/bigfred/architecture/16-dcc-bus/README.md) and the
[`hardware`](../../specs/hardware/README.md) bring-up guides.

## [RailBOX RB1110 / RB1110-Mini](./rb1110.md)

[RailBOX Electronics](https://www.railbox.pl/) — **`z21`** UDP on LAN/WiFi
(recommended).

## [Digikeijs DR5000](./dr5000.md)

[Digikeijs](https://www.digikeijs.com/) — **`loconet_serial`** + Uhlenbrock 63120
on **LocoNet-T**.
